# Álgebra Linear para Machine Learning

## Descrição

Este repositório contém a resolução de um exercício proposto no curso **Álgebra para Machine Learning e Data Science**.

O código **não é totalmente de minha autoria**. Foi fornecido um código-base pelo curso, no qual algumas partes estavam intencionalmente incompletas. Minha tarefa consistiu em **preencher os trechos em branco**, aplicando corretamente os conceitos abordados.

---

## Objetivo do Exercício

O exercício teve como objetivo reforçar o aprendizado em:

- Eliminação de Gauss
- Substituição para trás (back substitution)
- Manipulação de matrizes aumentadas
- Identificação de pivôs
- Aplicação de operações elementares em linhas

---

## Conteúdos Envolvidos

- Álgebra Linear
- Sistemas Lineares
- Algoritmos
- Operações matriciais
- Fundamentos matemáticos para Machine Learning

---

## Contexto Acadêmico

Atividade desenvolvida como parte do curso **Álgebra para Machine Learning e Data Science**, com foco no entendimento prático dos conceitos matemáticos utilizados em **Machine Learning** e **Data Science**.

## Observações

Apesar de o código não ter sido desenvolvido do zero, o exercício contribuiu significativamente para o aprendizado, pois exigiu análise do algoritmo, compreensão do fluxo lógico e aplicação correta dos conceitos matemáticos.
