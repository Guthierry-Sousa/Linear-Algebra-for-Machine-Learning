# Álgebra Linear para Machine Learning e Data Science

Este repositório tem como objetivo organizar e documentar estudos em **Álgebra Linear** com foco em aplicações práticas em **Machine Learning** e **Data Science**.

O material aqui apresentado busca estabelecer uma base matemática sólida, conectando conceitos teóricos fundamentais com sua utilização em algoritmos de aprendizado de máquina.

## Referências

As principais referências incluem livros clássicos de Álgebra Linear, cursos acadêmicos e materiais amplamente utilizados na área de Machine Learning e Data Science.
